プライバシーポリシー

このプライバシー ポリシーでは、WIFIGATE がウェブサイト、モバイル アプリ、WIFIGATE デバイス、コントローラー、招待状、アクセス イベント、および関連するアクセス制御機能に関連する情報を収集、使用、保存、表示、共有する方法について説明します。

発効日: [発効日] ウェブサイト運営者：イーツシステムテック
この文書は一般的なプライバシー ポリシーの草案として提供されており、公開前に資格のある法律顧問によるレビューが必要です。

1. 概要

このプライバシー ポリシーは、顧客、不動産所有者、管理者、設置者、ユーザー、ゲストが使用する WIFIGATE Web サイト、モバイル アプリ、WIFIGATE デバイス、コントローラー、および関連するアクセス制御機能に適用されます。

製品によってどのような情報が処理されるか、またその情報がどのように使用、保存、表示、共有されるかについて一般的な用語で説明されています。

2. このポリシーの対象範囲

このポリシーには、Web サイトへのアクセス、製品に関する問い合わせ、サポート コミュニケーション、モバイル アプリの使用、アクセス管理、招待、アクセス イベント、デバイスのステータス、技術的診断、および関連する製品の操作に関する情報が含まれます。

このポリシーは、特定の施設または管轄区域にも適用される顧客固有の契約、物件通知、設置者通知、またはサードパーティのポリシーに代わるものではありません。

3. WIFIGATE が収集する情報

使用されている製品機能に応じて、WIFIGATE は次のカテゴリの情報を処理する場合があります。

  • Web サイト、WhatsApp、電子メール、電話、またはその他のサポート チャネルを通じて送信された連絡先の詳細および問い合わせ情報。
  • 名前、電話番号、アプリ言語、アクセス関連の役割情報などのユーザー プロファイルの詳細。
  • 管理者、ユーザー、ゲスト、招待状、招待のタイミング、アクセス許可、関連するプロパティまたはグループの詳細などのアクセス管理情報。
  • アクセスイベント、アクセス履歴、デバイスステータス、技術診断、アップデート関連情報、製品ログ。
  • ローカル接続ステータス、権限、電話の互換性、構成の詳細など、製品の操作に必要なデバイスとアプリの情報。
  • 選択した言語やアクセシビリティの設定など、Web サイトの設定情報と同様のブラウザ側の設定。

4. WIFIGATE による情報の使用方法

WIFIGATE は次の目的で情報を使用することがあります。

  • Web サイト、アプリ、デバイス、および関連するアクセス制御機能を提供、運用、構成、保守、およびサポートするため。
  • ユーザー、管理者、ゲスト、招待、アクセス許可、製品設定を管理するため。
  • アクセス イベント、システム アクティビティ、デバイスのステータス、および関連製品情報を許可されたユーザーに表示するため。
  • 技術的な問題のトラブルシューティング、製品の信頼性の向上、安全性、構成、またはサポートの問題の確認を行うため。
  • お問い合わせへの対応、顧客サポートの提供、インストールに関する連絡、リクエストまたはアカウントに関連する製品またはサービス情報の送信のため。

5. 情報が共有される場合

WIFIGATE は、そのアクセスが製品の意図された操作の一部である場合、認可された不動産所有者、管理者、設置者、またはユーザーが情報を表示または利用できるようにする場合があります。

また、サポートの提供、技術的または安全性の問題への対処、法的義務の遵守、適用される条件の執行、または権利、財産、および安全の保護のために合理的に必要な場合には、情報が共有される場合もあります。

このポリシー草案には、個人情報の公的販売については記載されていません。製品データの共有慣行における今後の変更は、更新されたプライバシー通知に反映される必要があります。

6. ローカルストレージと製品固有のストレージ

実装に応じて、情報はモバイル デバイス上、WIFIGATE デバイス上、または製品で使用される別の製品固有の保存方法を使用してローカルに保存される場合があります。

インストールが異なれば、使用する製品設定やストレージの配置も異なる場合があります。顧客、不動産所有者、管理者は、自身のインストールとアクセスの実践を見直し、環境内でデータがどのように処理されるかを理解する必要があります。

7. ウェブサイトの設定、Cookie、および類似のテクノロジー

WIFIGATE Web サイトでは、選択した言語とアクセシビリティ設定を記憶し、サイトの使いやすさを向上させるために、設定ストレージなどの限られたブラウザ側テクノロジーを使用する場合があります。

Web サイトの Cookie および同様のテクノロジーの詳細については、WIFIGATE Cookie ページもご覧ください。

8. データの保持

WIFIGATE は、製品の操作、サポート、トラブルシューティング、法令順守、紛争解決、記録保持、またはその他の正当な製品またはビジネス目的に合理的に必要な期間、情報を保持することがあります。

保持期間は、情報の種類、インストール、使用される機能、適用される法的義務、および情報が製品管理またはセキュリティレビューに引き続き必要かどうかによって異なります。

9. セキュリティ

WIFIGATE は、情報と製品へのアクセスを保護することを目的とした合理的な措置を講じていますが、Web サイト、アプリ、デバイス、電話、ワイヤレス接続、またはアクセス制御システムが完全に安全であることを保証することはできません。

顧客、管理者、設置者、ユーザーも、デバイス、権限、招待、アクセス方法、インストール設定を保護することで、セキュリティにおいて重要な役割を果たします。

10. あなたの選択と権利

製品の設定および適用される法律によっては、WIFIGATE が取り扱う特定の個人データへのアクセス、修正、削除、制限、またはその情報を要求できる場合があります。

一部のリクエストは、インストールおよび関連情報を誰が管理しているかに応じて、関連する不動産所有者、システム管理者、設置者、またはアカウント担当者による処理が必要になる場合があります。

11.子供たち

WIFIGATE は子供向け製品として設計されていません。お子様に関する個人情報が不適切に提供されたと思われる場合は、WIFIGATE に連絡して問題を確認してください。

12. このプライバシーポリシーの変更

WIFIGATE は、このプライバシー ポリシーを随時更新することがあります。最新バージョンは、このページに公開された時点、または製品とともに提供された時点で適用されます。
