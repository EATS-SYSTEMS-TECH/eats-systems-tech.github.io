このページでは、WIFIGATE のウェブサイトが Cookie、ブラウザーのローカルストレージ、および類似技術をどのように使用する場合があるかを説明します。

1. 概要

このページでは、WIFIGATE Web サイトが Cookie および同様のブラウザ側テクノロジーをどのように使用するかについて説明します。

現在の Web サイトの実装では、WIFIGATE は主に、言語やアクセシビリティ設定などの Web サイト機能をサポートするために、限られた設定ストレージと関連するブラウザー テクノロジを使用します。

2. 現在のウェブサイトで使用されているもの

現在の Web サイトの実装に基づいて、WIFIGATE は限られたブラウザ側のストレージを使用して、選択した言語やアクセシビリティの選択肢などの訪問者の好みを記憶します。

これにより、Web サイトは訪問間で選択した表示設定を維持し、手動で変更を繰り返すことなくユーザー エクスペリエンスを向上させることができます。

3. Cookie および類似技術の種類

Web サイトは、次のカテゴリーのブラウザー側テクノロジーに依存している場合があります。

  • Web サイトの基本的な運用、セキュリティ、アクセシビリティ機能に必要な必須テクノロジー。
  • 訪問者が選択した言語選択とアクセシビリティ設定を記憶するために使用される環境設定ストレージ。
  • フォントやコンテンツ配信サービスなど、Web サイトで使用されるサードパーティプロバイダーに接続された技術的なリクエストまたはブラウザー側のリソース。

4. Preference Storage および類似のテクノロジー

現在の Web サイトの実装では、選択された言語設定とアクセシビリティ設定にローカル ブラウザー ストレージが使用されます。

ローカル ブラウザのストレージは従来の Cookie と必ずしも同じではありませんが、訪問者のデバイスまたはブラウザの設定を記憶することで同様の目的を果たすことができます。

5. サードパーティのサービスと技術的なリクエスト

Web サイトは、ページの読み込み時に標準の技術リクエスト情報を受け取るフォント サービスなどのサードパーティのリソースを読み込む場合があります。

WIFIGATE は、それらのサードパーティプロバイダーのポリシーを制御しません。独自の規約、プライバシー通知、および技術的慣行も適用される場合があります。

6. 広告および分析 Cookie に関する現在の立場

現在の Web サイトの実装に基づいて、WIFIGATE はこの Web サイトに独自の広告 Cookie や分析 Cookie を意図的に展開しません。

将来的に Web サイトの慣行が変更された場合は、このページを更新して、サイトで使用される新しい分析、測定、広告、または同意関連のテクノロジーについて説明する必要があります。

7. ブラウザ設定の管理

通常、Cookie、ローカル ストレージ、および類似のテクノロジは、ブラウザの設定、ブラウザのプライバシー ツール、またはデバイスの設定を通じて制御できます。

一部のテクノロジーをブロックまたはクリアすると、言語設定、アクセシビリティ設定、または Web サイトのエクスペリエンスの一部に影響を与える可能性があります。
