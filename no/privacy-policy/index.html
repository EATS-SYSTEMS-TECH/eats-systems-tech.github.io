Personvernerklæring

Denne personvernerklæringen forklarer hvordan WIFIGATE kan samle inn, bruke, lagre, vise og dele informasjon knyttet til nettstedet, mobilappen, WIFIGATE-enheten, kontrolleren, invitasjoner, tilgangsbegivenheter og relaterte tilgangskontrollfunksjoner.

Ikrafttredelsesdato: [Ikrafttredelsesdato] Nettstedseier: EATS SYSTEMS TECH
Dette dokumentet er gitt som et generelt utkast til personvernregler og bør gjennomgås av kvalifisert juridisk rådgiver før publisering.

1. Oversikt

Denne personvernerklæringen gjelder WIFIGATE-nettstedet, mobilappen, WIFIGATE-enheten, kontrolleren og relaterte tilgangskontrollfunksjoner som brukes av kunder, eiendomseiere, administratorer, installatører, brukere og gjester.

Den forklarer i generelle termer hvilken informasjon som kan håndteres av produktet og hvordan denne informasjonen kan brukes, lagres, vises og deles.

2. Hva denne forsikringen dekker

Denne policyen dekker informasjon relatert til nettstedbesøk, produktforespørsler, støttekommunikasjon, mobilappbruk, tilgangsadministrasjon, invitasjoner, tilgangshendelser, enhetsstatus, teknisk diagnostikk og relaterte produktoperasjoner.

Denne policyen erstatter ikke kundespesifikke avtaler, eiendomsmeldinger, installatørmeldinger eller tredjepartspolicyer som også kan gjelde i en bestemt installasjon eller jurisdiksjon.

3. Informasjon WIFIGATE kan samle inn

Avhengig av produktfunksjonene som brukes, kan WIFIGATE håndtere følgende informasjonskategorier:

  • Kontaktdetaljer og forespørselsinformasjon sendt inn via nettstedet, WhatsApp, e-post, telefon eller andre støttekanaler.
  • Brukerprofildetaljer som navn, telefonnummer, appspråk og tilgangsrelatert rolleinformasjon.
  • Tilgangsadministrasjonsinformasjon som administratorer, brukere, gjester, invitasjoner, invitasjonstidspunkt, tilgangstillatelser og relaterte eiendoms- eller gruppedetaljer.
  • Få tilgang til hendelser, tilgangshistorikk, enhetsstatus, teknisk diagnostikk, oppdateringsrelatert informasjon og produktlogger.
  • Enhets- og appinformasjon som trengs for produktdrift, for eksempel lokal tilkoblingsstatus, tillatelser, telefonkompatibilitet og konfigurasjonsdetaljer.
  • Informasjon om nettstedspreferanser og lignende innstillinger på nettleseren, inkludert utvalgte språk- og tilgjengelighetspreferanser.

4. Hvordan WIFIGATE kan bruke informasjon

WIFIGATE kan bruke informasjon til følgende formål:

  • For å tilby, betjene, konfigurere, vedlikeholde og støtte nettstedet, appen, enheten og relaterte tilgangskontrollfunksjoner.
  • For å administrere brukere, administratorer, gjester, invitasjoner, tilgangstillatelser og produktinnstillinger.
  • For å vise tilgangshendelser, systemaktivitet, enhetsstatus og relatert produktinformasjon til autoriserte brukere.
  • For å feilsøke tekniske problemer, forbedre produktets pålitelighet og gjennomgå sikkerhets-, konfigurasjons- eller støtteproblemer.
  • For å svare på forespørsler, gi kundestøtte, kommunisere om installasjoner og sende produkt- eller tjenesteinformasjon som er relevant for en forespørsel eller konto.

5. Når informasjon kan deles

WIFIGATE kan vise eller gjøre informasjon tilgjengelig for autoriserte eiendomseiere, administratorer, installatører eller brukere der denne tilgangen er en del av produktets tiltenkte drift.

Informasjon kan også deles der det er rimelig nødvendig for å gi støtte, løse tekniske eller sikkerhetsmessige problemer, overholde juridiske forpliktelser, håndheve gjeldende vilkår eller beskytte rettigheter, eiendom og sikkerhet.

Dette policyutkastet beskriver ikke noe offentlig salg av personlig informasjon. Eventuelle fremtidige endringer i praksis for deling av produktdata bør gjenspeiles i en oppdatert personvernerklæring.

6. Lokal lagring og produktspesifikk lagring

Avhengig av implementeringen, kan informasjon lagres lokalt på mobilenheten, på WIFIGATE-enheten eller ved å bruke en annen produktspesifikk lagringsmetode som brukes av produktet.

Ulike installasjoner kan bruke ulike produktinnstillinger eller oppbevaringsarrangement. Kunder, eiendomseiere og administratorer bør gjennomgå sin egen installasjon og tilgangspraksis for å forstå hvordan data håndteres i deres miljø.

7. Nettstedspreferanser, informasjonskapsler og lignende teknologier

WIFIGATE-nettstedet kan bruke begrensede nettlesersideteknologier, inkludert preferanselagring, for å huske utvalgte språk- og tilgjengelighetsinnstillinger og for å forbedre nettstedets brukervennlighet.

Se også WIFIGATE-informasjonskapsler-siden for mer detaljer om nettsideinformasjonskapsler og lignende teknologier.

  • Informasjonskapsler

8. Oppbevaring av data

WIFIGATE kan beholde informasjonen så lenge det er rimelig nødvendig for produktdrift, støtte, feilsøking, juridisk overholdelse, tvisteløsning, journalføring eller andre legitime produkt- eller forretningsformål.

Oppbevaringsperioder kan variere avhengig av typen informasjon, installasjonen, funksjonen som brukes, gjeldende juridiske forpliktelser og om informasjonen fortsatt er nødvendig for produktadministrasjon eller sikkerhetsgjennomgang.

9. Sikkerhet

WIFIGATE bruker rimelige tiltak for å beskytte informasjon og produkttilgang, men ingen nettside, app, enhet, telefon, trådløs tilkobling eller tilgangskontrollsystem kan garanteres helt sikkert.

Kunder, administratorer, installatører og brukere spiller også en viktig rolle i sikkerheten ved å beskytte enheter, tillatelser, invitasjoner, tilgangsmetoder og installasjonsinnstillinger.

10. Dine valg og rettigheter

Avhengig av produktoppsettet og gjeldende lov, kan du kanskje be om tilgang til, retting av, sletting av, begrensning av eller informasjon om visse personopplysninger som håndteres av WIFIGATE.

Noen forespørsler må kanskje håndteres av den aktuelle eiendomseieren, systemadministratoren, installatøren eller kontokontakten, avhengig av hvem som kontrollerer installasjonen og relatert informasjon.

11. Barn

WIFIGATE er ikke designet som et barneprodukt. Hvis personopplysninger knyttet til et barn antas å ha blitt gitt upassende, kontakt WIFIGATE slik at saken kan vurderes.

12. Endringer i denne personvernerklæringen

WIFIGATE kan oppdatere denne personvernerklæringen fra tid til annen. Den siste versjonen vil gjelde når den publiseres på denne siden eller på annen måte gjøres tilgjengelig med produktet.
